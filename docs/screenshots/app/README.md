# Screenshots do aplicativo

Capturas usadas no [README](../../../README.md), extraídas da **listagem pública do LodgeFlow na App
Store** — ou seja, material de divulgação já publicado, sem dados privados.

| Arquivo | Tela |
|---|---|
| `planner.png` | Planner semanal |
| `dia.png` | Visão do dia e progresso |
| `planejar-ia.png` | Assistente de planejamento com IA |
| `financas.png` | Painel financeiro |
| `pomodoro.png` | Modo foco / Pomodoro |

As imagens são recortadas para conter apenas o card de divulgação, sem a barra de status do
dispositivo nem elementos da interface da App Store.

## Ao adicionar ou substituir capturas

Se a captura **não** vier do material público da loja, confirme antes que ela não contém:

- Dados pessoais reais — nomes, e-mails, telefones, endereços
- Valores financeiros verdadeiros
- Tokens, chaves ou identificadores visíveis em telas de depuração
- Endpoints, hosts ou URLs internas na barra de endereço ou em console
- Informação de conta de terceiros

**Use uma conta de demonstração com dados fictícios.** É mais confiável do que revisar cada captura
depois.

## Recomendações técnicas

- **Formato:** PNG
- **Proporção:** mantenha a mesma entre as imagens, para o grid do README ficar alinhado
- **Resolução:** capture em 2x/3x e deixe o Markdown redimensionar
- **Tamanho:** comprima antes de subir; mantenha cada arquivo abaixo de ~500 KB
- **Tema:** prefira consistência visual entre as capturas
